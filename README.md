# 💕 Hey You — Romantic Quiz Game

A beautiful, cinematic quiz game to learn everything about your special someone — built with love, humor, and way too many heart emojis.

## ✨ Features
- 🎬 Cinematic intro with typewriter effect
- 🌌 Starfield + floating hearts background
- 💎 Glassmorphism cards with 3D transitions
- 🎮 Level system (The Basics → Final Boss)
- 📧 EmailJS integration to receive answers
- 📱 Fully responsive (mobile-friendly)
- ⌨️ Keyboard navigation support

## 🚀 Setup
1. Clone the repo
2. Open `index.html` in browser
3. Connect EmailJS (see below)

## 📧 EmailJS Setup
1. Create account at [emailjs.com](https://emailjs.com)
2. Add your email service
3. Create template with variable `{{message}}`
4. Fill in `EMAIL_CONFIG` in `script.js`
5. Uncomment the EmailJS send block
